Some general information about how to use the TTS Editor

All regular shortcuts available through your right mouse click are available.
The TTS Editor offers an AudioEdit functionality, which as of now relies on Audacity, which must be installed on your machine. This AudioEdit functionality allows for manipulating the wave form produced by the synthesis and also allows for saving the resulting file.
Please note, that the tuning via the editor is unaffected by the waveform tuning. Therefore, this should be the last step in the tuning process! If you save the data in the editor, it will not reflect changes done via the AudioEdit functionality.
